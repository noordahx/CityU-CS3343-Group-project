+CS3343 Group project
+Room Booking System
+Installation Guide
+
+1. Install Java 11 or higher (https://www.oracle.com/java/technologies/downloads/)
+2. Add JAVA to path
+3. Double-click the jar file to run the program (or run the command "java -jar CS3343-Group-project.jar")
+4. Enjoy!
+5. Before closing the program, please save your data by choosing save directory and save file name
+6. If you want to load the data, please choose the directory and file name that you saved before
